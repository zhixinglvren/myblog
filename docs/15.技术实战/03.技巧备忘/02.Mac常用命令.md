## 1 Mac新建软连接

**Mac根目录下无法新建文件夹，可通过软连接方式实现效果。**

````shell
zhixinglvren@zhixinglvrendeMacBook-Pro blog % sudo vi /etc/synthetic.conf

data    /Users/data
img     /Users/zhixinglvren/githubprojects/blog/docs/.vuepress/public/img
````
:::tip
`synthetic.conf` 文件中 `key` 和 `value` 之间是 `TAB` 而不是 `空格` 。 
:::


`重启电脑生效`

## 2 配置环境变量

`.bash_profile` 没有编辑权限，给自己的用户设置编辑权限，执行以下命令：
````shell
/usr/bin/sudo /bin/chmod u+rwx /Users/zhixinglvren/.bash_profile
/usr/bin/sudo /usr/sbin/chown zhixinglvren /Users/zhixinglvren/.bash_profile

# zhixinglvren指自己的用户名
````

编辑配置文件
````shell
vi ~/.bash_profile
````

配置生效
````shell
source ~/.bash_profile
````
